********** MÉTODO **********
É uma porção de código (sub-rotina) que é disponibilizada por uma classe. Este
é executado quando é feita uma requisição a ele. São responsáveis por definir e realizar
um determinado comportamento.

Exemplo:
public static RETORNO NOME(PARÂMETROS) {...CORPO}

Assinatura: é a forma de identificar unicamente o método.
Assinatura = nome + parâmetros

Construtor e destrutor: são métodos especiais usados na orientação a objetos.

Mensagem: é o ato de solicitar ao método que o mesmo execute. Esta pode ser direcionada a um objeto ou a uma classe.

O nome de um método deve ser descritivo e curto, mostrando melhor o seu significado.

Sobrecarga: é a capacidade de definir métodos para diferentes contextos, mas preservando seu nome.
Sobrecarga x sobrescrita

Retorno:
É uma instrução de interrupção
Simbologia: return
O tipo de retorno do método é definido na sua criação e pode ser um tipo primitivo ou objeto.
